# rtabmap_ros_my
    
自作 Turtlebot3(foxbot_core3.ino) で、自作 Stereo Camera , OAK-D-Lite を使って    
rtabmap_ros が実行できる、 launch ファイル。    
    
1. インストール方法    
$ cd ~catkin_ws/src    
$ git clone https://github.com/tosa-no-onchan/rtabmap_ros_my.git    
$ cd ..    
$ catkin_make
